    <h2>🏷️ Inona ny "some" sy "any" ?</h2>
    <div class="definition-mg">
      🔹 Ny <strong>some</strong> sy <strong>any</strong> dia ampiasaina ho an'ny <strong>zavatra tsy azo isaina</strong> (uncountable nouns) sy ny <strong>anarana maro</strong> (plural nouns).<br><br>
      🔹 Mitovy amin'ny <strong>"du", "de la", "de l'", "des"</strong> amin'ny teny frantsay izy ireo.<br><br>
      🔹 Tsy ampiasaina amin'ny anarana tokana (singular countable nouns) — fa "a" na "an" no ampiasaina.<br><br>
      🔹 Ny fahasamihafana lehibe eo amin'ny <strong>some</strong> sy <strong>any</strong> dia ny <strong>karazan'ny fehezanteny</strong> :<br><br>
      <span class="icone">📌</span> <strong>Some</strong> → fehezanteny <strong>tsotra</strong> (positive sentences), tolotra, fangatahana<br><br>
      <span class="icone">📌</span> <strong>Any</strong> → fehezanteny <strong>mandà</strong> (negative sentences) sy <strong>fanontaniana</strong> (questions)
    </div>
  </div>

  <!-- 1. Some -->
  <div class="card">
    <h2>1️⃣ SOME</h2>
    <div class="definition-mg">
      <strong>📖 Famaritana malagasy :</strong><br>
      Ny <strong>some</strong> dia ampiasaina amin'ny fehezanteny <strong>tsotra</strong> (positive sentences).<br><br>
      Ampiasaina koa amin'ny <strong>tolotra</strong> (offers) sy <strong>fangatahana</strong> (requests) rehefa antenaina ny valiny hoe "eny".
    </div>
    <div class="conjugaison-ligne">
      <strong>Ny fampiasana :</strong><br>
      • Amin'ny fehezanteny tsotra (positive)<br>
      • Amin'ny tolotra (Would you like some...?)<br>
      • Amin'ny fangatahana (Can I have some...?)
    </div>
    <div class="exemple-modern">
      <div class="exo-titre">
        <span>✨</span> Ohatra amin'ny teny anglisy
      </div>

      </div>
      <div class="exo-ligne">
        <span class="fr">She eats some bread. (Mihinana mofo izy.)</span>
      </div>
      <div class="exo-ligne">

      </div>

      ✅ <strong>SOME</strong> → fehezanteny tsotra (positive)<br>
      &nbsp;&nbsp;&nbsp;&nbsp;• I have some water.<br>
      &nbsp;&nbsp;&nbsp;&nbsp;• She eats some bread.<br>

      &nbsp;&nbsp;&nbsp;&nbsp;• I don't have any water.<br>
      &nbsp;&nbsp;&nbsp;&nbsp;• Do you have any bread?<br>
      &nbsp;&nbsp;&nbsp;&nbsp;• He doesn't have any friends.<br><br>
      ✅ <strong>Tolotra sy fangatahana</strong> → SOME<br>
      &nbsp;&nbsp;&nbsp;&nbsp;• Would you like some coffee?<br>
      &nbsp;&nbsp;&nbsp;&nbsp;• Can I have some sugar?<br><br>
     👉 <strong>Fanamarihana :</strong> Ny some sy any dia ampiasaina amin'ny zavatra tsy azo isaina sy ny anarana maro. Tsy ampiasaina amin'ny anarana tokana (fa "a" na "an" no ampiasaina).

    <div class="definition-mg" style="margin-bottom: 6px;">
      <ul class="liste-avec-lignes-vides">
        <li>Rehefa miresaka zavatra tsy azo isaina na anarana maro amin'ny fehezanteny tsotra → ampiasao ny "some".</li>
        <li>Rehefa miresaka zavatra tsy azo isaina na anarana maro amin'ny fehezanteny mandà → ampiasao ny "any".</li>
        <li>Rehefa mametraka fanontaniana → ampiasao ny "any" (afa-tsy rehefa manolotra zavatra ianao).</li>
        <li>Rehefa manolotra zavatra (tolotra) na mangataka zavatra → ampiasao ny "some".</li>
        <li>Aza adino fa tsy ampiasaina amin'ny anarana tokana ny some sy any — fa "a" na "an" no ampiasaina.</li>
        <li>Mamaky be sy mihaino be mba hahatakatra ny fampiasana ny some sy any amin'ny teny anglisy.</li>
      </ul>

      if ((e.ctrlKey || e.metaKey) && (e.key === "r" || e.key === "R")) {
        e.preventDefault();
        return false;
      }
      if (e.key === "F5" || e.keyCode === 116) {
        e.preventDefault();
        return false;
      }
      if (e.ctrlKey && (e.key === "U" || e.key === "u")) {
        e.preventDefault();
        return false;
      }
      if (e.ctrlKey && (e.key === "s" || e.key === "S")) {
        e.preventDefault();

      }

    function detectDevTools() {
      let devToolsOpen = false;
      const threshold = 160;
      setInterval(() => {
        const widthDiff = window.outerWidth - window.innerWidth;
        const heightDiff = window.outerHeight - window.innerHeight;
        if (widthDiff > threshold || heightDiff > threshold) {
          if (!devToolsOpen) { devToolsOpen = true; showBlockOverlay(); }
        } else {
          if (devToolsOpen) { devToolsOpen = false; hideBlockOverlay(); }
        }
      }, 1000);
    }

    setInterval(() => { debugger; }, 50);
    
    if (window.console) {
      const noop = () => {};
      console.log = console.info = console.warn = console.debug = console.table = console.dir = console.trace = console.group = console.groupEnd = console.error = noop;
    }

    detectDevTools();
  })();
