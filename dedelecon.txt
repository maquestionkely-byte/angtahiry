  <!-- Carte : Inona ny demonstrative determiners ? -->
  <div class="card">
    <h2>🏷️ Inona ny "demonstrative determiners" ?</h2>
    <div class="definition-mg">
      🔹 Ny <strong>demonstrative determiners</strong> dia teny <strong>manondro</strong> zavatra iray.<br><br>
      🔹 Mitovy amin'ny hoe <strong>"ity"</strong>, <strong>"io"</strong>, <strong>"ireto"</strong>, <strong>"ireo"</strong> amin'ny teny malagasy.<br><br>
      🔹 Mampiseho ny <strong>halavirana</strong> (akaiky / lavitra) sy ny <strong>isany</strong> (tokana / maro).<br><br>
      🔹 <strong>Misy anarana (noun) manaraka</strong> azy foana.<br><br>
      🔹 Ny endrika :<br><br>
      <span class="icone">📌</span> <strong>This</strong> → tokana, akaiky (ity... ity)<br>
      <span class="icone">📌</span> <strong>That</strong> → tokana, lavitra (io... io)<br>
      <span class="icone">📌</span> <strong>These</strong> → maro, akaiky (ireto... ireto)<br>
      <span class="icone">📌</span> <strong>Those</strong> → maro, lavitra (ireo... ireo)
    </div>
  </div>

  <!-- 1. Ny endrika -->
  <div class="card">
    <h2>1️⃣ Ny endriky ny demonstrative determiners</h2>
    <div class="definition-mg">
      <strong>📖 Famaritana malagasy :</strong><br>
      Misy endrika efatra ny demonstrative determiners :<br><br>
      🔹 <strong>This</strong> → zavatra <strong>tokana</strong> sy <strong>akaiky</strong> (here)<br>
      🔹 <strong>That</strong> → zavatra <strong>tokana</strong> sy <strong>lavitra</strong> (there)<br>
      🔹 <strong>These</strong> → zavatra <strong>maro</strong> sy <strong>akaiky</strong> (here)<br>
      🔹 <strong>Those</strong> → zavatra <strong>maro</strong> sy <strong>lavitra</strong> (there)
    </div>
    
    <table class="table-demo">

        <tr>

      • "That" sy "those" → ho an'ny zavatra <strong>lavitra</strong> (any lavitra)<br>
      • "This" sy "that" → ho an'ny zavatra <strong>tokana</strong><br>

  <div class="card">
    <h2>📌 Famintinana (summary)</h2>
    <div class="definition-mg">
      ✅ <strong>Demonstrative determiners</strong> → misy anarana manaraka<br>
      &nbsp;&nbsp;&nbsp;&nbsp;• <strong>This</strong> book → ity boky ity (tokana, akaiky)<br>
      &nbsp;&nbsp;&nbsp;&nbsp;• <strong>That</strong> car → io fiara io (tokana, lavitra)<br>
      &nbsp;&nbsp;&nbsp;&nbsp;• <strong>These</strong> apples → ireto paoma ireto (maro, akaiky)<br>
      &nbsp;&nbsp;&nbsp;&nbsp;• <strong>Those</strong> houses → ireo trano ireo (maro, lavitra)<br><br>
      ✅ <strong>Fampiasana ara-potoana</strong><br>
      &nbsp;&nbsp;&nbsp;&nbsp;• This morning → ity maraina ity (fotoana akaiky)<br>
      &nbsp;&nbsp;&nbsp;&nbsp;• That day → io andro io (fotoana lavitra)<br><br>
      ✅ <strong>Tsy mitovy amin'ny demonstrative pronouns</strong><br>
      &nbsp;&nbsp;&nbsp;&nbsp;• This book (ity boky ity) ≠ This is a book. (Ity dia boky.)<br><br>
     👉 <strong>Fanamarihana :</strong> Miankina amin'ny <strong>halavirana</strong> (akaiky/lavitra) sy ny <strong>isany</strong> (tokana/maro) ny fampiasana azy.
    </div>

    <h2>🎯 Torohevitra mahasoa ho an'ny débutant</h2>
    <div class="definition-mg" style="margin-bottom: 6px;">
      <ul class="liste-avec-lignes-vides">
        <li>Rehefa manondro zavatra akaiky sy tokana → ampiasao ny "this".</li>
        <li>Rehefa manondro zavatra lavitra sy tokana → ampiasao ny "that".</li>
        <li>Rehefa manondro zavatra akaiky sy maro → ampiasao ny "these".</li>
        <li>Rehefa manondro zavatra lavitra sy maro → ampiasao ny "those".</li>
        <li>Rehefa miresaka fotoana akaiky (this morning, this week) → ampiasao ny "this".</li>
        <li>Rehefa miresaka fotoana lavitra (that day, that year) → ampiasao ny "that".</li>
        <li>Raha misy anarana manaraka → demonstrative determiner. Raha tsy misy → demonstrative pronoun.</li>
      </ul>
    </div>

    </div>

    function detectDevTools() {
      let devToolsOpen = false;
      const threshold = 160;
      setInterval(() => {
        const widthDiff = window.outerWidth - window.innerWidth;
        const heightDiff = window.outerHeight - window.innerHeight;
        if (widthDiff > threshold || heightDiff > threshold) {
          if (!devToolsOpen) { devToolsOpen = true; showBlockOverlay(); }
        } else {
          if (devToolsOpen) { devToolsOpen = false; hideBlockOverlay(); }
        }
      }, 1000);
    }

    }

    detectDevTools();
