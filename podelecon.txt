  <div class="card">
    <h2>🏷️ Inona ny "possessive determiners" ?</h2>
    <div class="definition-mg">
      🔹 Ny <strong>possessive determiners</strong> dia teny milaza hoe <strong>an'iza</strong> ny zavatra iray.<br><br>
      🔹 Mitovy amin'ny hoe <strong>"-ko"</strong>, <strong>"-nao"</strong>, <strong>"-ny"</strong>, <strong>"-ntsika"</strong>, <strong>"-nareo"</strong>, <strong>"-ny"</strong> (maro) amin'ny teny malagasy.<br><br>
      🔹 <strong>Misy anarana (noun) manaraka</strong> azy foana.<br><br>
      🔹 Miankina amin'ny <strong>tompony</strong> (owner) ny possessive, fa tsy amin'ny zavatra.<br><br>
      🔹 Ny endrika :<br><br>
      <span class="icone">📌</span> <strong>My</strong> → -ko (ny ahy)<br>
      <span class="icone">📌</span> <strong>Your</strong> → -nao (ny anao)<br>
      <span class="icone">📌</span> <strong>His / Her / Its</strong> → -ny (ny any / ny azy)<br>
      <span class="icone">📌</span> <strong>Our</strong> → -ntsika / -nay (ny antsika / ny anay)<br>
      <span class="icone">📌</span> <strong>Your</strong> → -nareo (ny anareo)<br>
      <span class="icone">📌</span> <strong>Their</strong> → -ny (ny azy ireo)
    </div>
  </div>

  <!-- Carte Torohevitra -->
  <div class="card">
    <h2>🎯 Torohevitra mahasoa ho an'ny débutant</h2>
    <div class="definition-mg" style="margin-bottom: 6px;">
      <ul class="liste-avec-lignes-vides">
        <li>Rehefa milaza fananana miaraka amin'ny anarana → ampiasao ny possessive determiner (my, your, his, her, its, our, their).</li>
        <li>Rehefa milaza fananana nefa tsy misy anarana → ampiasao ny possessive pronoun (mine, yours, his, hers, ours, theirs).</li>
        <li>Aza adino fa "its" (tsy misy apostrophe) dia possessive determiner, fa "it's" dia "it is".</li>
        <li>Tsy miova na lahy na vavy ny zavatra : my father (raiko) / my mother (reniko).</li>
        <li>Vakio mafy ny ohatra isan'andro mba hahatsapanao ny fampiasana azy.</li>
        <li>Mamaky be sy mihaino be mba hahatakatra ny fahasamihafana eo amin'ny determiners sy pronouns.</li>
      </ul>
    </div>
    <div class="highlight" style="margin-top: 8px;">
      🌱 <strong>Tsarovy :</strong> Ny fianarana possessive determiners dia tena ilaina satria isaky ny miteny na manoratra anglisy ianao dia mampiasa azy. Mianara tsikelikely ary aza matahotra diso !
    </div>

    function detectDevTools() {
      let devToolsOpen = false;
      const threshold = 160;
      setInterval(() => {
        const widthDiff = window.outerWidth - window.innerWidth;
        const heightDiff = window.outerHeight - window.innerHeight;
        if (widthDiff > threshold || heightDiff > threshold) {
          if (!devToolsOpen) { devToolsOpen = true; showBlockOverlay(); }
        } else {
          if (devToolsOpen) { devToolsOpen = false; hideBlockOverlay(); }
        }
      }, 1000);
    }
